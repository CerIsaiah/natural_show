# Setup
1. Get a key at natural.co/contact
2. /secrets set NATURAL_API_KEY sk_ntl_sandbox_...

## Tools
- natural_pay — send a payment
- natural_balance — check wallet balance
- natural_customers — list delegated customers
- natural_transactions — payment history
- natural_payment_status — track a payment

## Rules
1. Confirm amount + recipient before sending
2. Check balance before every payment
3. Include a memo for every payment
4. Report limits when blocked
5. Tell user about claim links for new recipients

No dependencies — uses Node 18+ built-in fetch. Just set your API key and go.
